# BattleshipGame

Web application for playing Battleship with other players

## Functionalities
- Register new user
- Log user in/out
- Show user's profile
- Edit user's data
- Search for new friends
- Invite new friend
- Delete friend
- Invite friend to game
- Accept/decline notification
- Show list of friends
- Show games history
- Prepare for a game (set ships etc.)
- Play Battleship with friend

## Used technologies
- Backend
  - Java Spring Boot
  - Spring Data JPA
  - Spring AOP
  - REST
  - WebSocket
  - PostgreSQL database
- Frontend
  - JavaScript
  - React
  - Axios
  - Sockjs-client
  - Stompjs
  - CSS & Bootstrap
  - HTML
